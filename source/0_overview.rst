PROJECT OVERVIEW
================

ESP32-S3 Uno Development Board retaines the formfactor of Arduino Uno board. This is a two-layer
board with the ESP32-S3 WROOM module at the top. The power and USB-C connection port, pin heards 
along the sides are located on the same side. The board also features the two control buttons, "BOOT"
and "EN", power-LED indicator, barrel jack connector for external power supply, a 3.3V voltage regulator
with reverse voltage protection and a power switch.
